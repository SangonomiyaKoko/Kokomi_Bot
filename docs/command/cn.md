## `/link` 指令说明

**用于绑定游戏账号，可通过 UID 或“服务器+IGN” 的方式完成绑定。**

### 🧩 指令格式

```bash
/link user:<用户标识>
```

### 🔧 参数说明

| 参数名 | 是否必填 | 说明                                                    |
| ------ | -------- | ------------------------------------------------------- |
| `user` | ✅       | 可输入游戏 UID，或使用格式 `服务器 IGN`（空格分隔）绑定 |

### 📌 使用示例

```bash
# 示例1：通过 UID 绑定
/link 2023619512

# 示例2：通过服务器和 IGN 绑定
/link asia SangonomiyaKokomi_
```

### 💡 注意事项

- 若 IGN 无法绑定成功，可尝试使用 UID 绑定方式。
- 国服用户如 IGN 中包含空格或特殊字符，推荐使用 **UID 方式** 进行绑定，以避免识别错误。

| `lang` | ✅       | 可选值：`cn`、`en`、`ja` |

### 📌 使用示例

```bash
/lang en
```

## `/algo` 指令说明

**用于切换用户评分算法。**

目前支持两种算法：

- `default`：默认评分算法。
- `none`：关闭评分功能，不再显示评分。

### 🧩 指令格式

```bash
/algo mode:<评分算法>
```

### 🔧 参数说明

| 参数名 | 是否必填 | 说明                     |
| ------ | -------- | ------------------------ |
| `mode` | ✅       | 可选值为 `default` 或 `none` |

### 📌 使用示例

```bash
/algo default
/algo none
```

---

## `/mode` 指令说明

**用于切换 Bot 返回图片的显示模式（日间 / 夜间）。**

目前支持两种模式：

- `light`：日间模式，亮色背景。
- `dark`：夜间模式，暗色背景。

### 🧩 指令格式

```bash
/mode mode:<显示模式>
```

### 🔧 参数说明

| 参数名 | 是否必填 | 说明                     |
| ------ | -------- | ------------------------ |
| `mode` | ✅       | 可选值为 `light` 或 `dark` |

### 📌 使用示例

```bash
/mode dark
/mode light
```

---

## `/theme` 指令说明

**用于切换 Bot 返回图片的视觉主题风格，或查看可用主题列表。**

当前支持两种用法：

1. 使用 `list` 参数查看所有可选主题及其代码。
2. 指定主题代码以切换当前主题。

### 🧩 指令格式

```bash
/theme list
# 或
/theme theme:<主题代码>
```

### 🔧 参数说明

| 参数名 | 是否必填 | 说明                     |
| ------ | -------- | ------------------------ |
| `theme` | ✅       | 输入主题代码, 使用 `list` 查看可选主题 |

### 📌 使用示例

```bash
# 查看当前可选主题列表
/theme list

# 切换为主题 mygo
/theme mygo
```
