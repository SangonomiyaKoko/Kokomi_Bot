## `/cls` 指令说明

**用于清除 Bot 运行过程中产生的不必要缓存文件。**

> 🔐 仅限具有管理员权限的用户使用。

### 🧩 指令格式

```bash
/cls
```

### 🔧 参数说明

无参数

### 📌 使用示例

```bash
/cls
```

### 💡 注意事项

- 清除操作不可恢复。

无参数

### 📌 使用示例

```bash
/admin
```

## `/link` 指令说明

**用于绑定游戏账号，可通过 UID 或“服务器+IGN” 的方式完成绑定。**

### 🧩 指令格式

```bash
/link user:<用户标识>
```

### 🔧 参数说明

| 参数名 | 是否必填 | 说明                                                    |
| ------ | -------- | ------------------------------------------------------- |
| `user` | ✅       | 可输入游戏 UID，或使用格式 `服务器 IGN`（空格分隔）绑定 |

### 📌 使用示例

```bash
# 示例1：通过 UID 绑定
/link 2023619512

# 示例2：通过服务器和 IGN 绑定
/link asia SangonomiyaKokomi_
```

### 💡 注意事项

- 若 IGN 无法绑定成功，可尝试使用 UID 绑定方式。
- 国服用户如 IGN 中包含空格或特殊字符，推荐使用 **UID 方式** 进行绑定，以避免识别错误。

## `/alias` 指令说明

**用于管理用户的游戏账号别名列表，简化后续查询操作。**

支持新增、删除、查看别名。设置后可通过别名代替 UID 或 IGN 快速查询战绩等信息。

### 🧩 指令格式

```bash
# 查看
/alias list
# 删除
/alias del index:<索引>
# 添加
/alias add alias:<别名> player:<用户标识>
```

### 🔧 参数说明

| 参数名   | 是否必填 | 说明                                                    |
| -------- | -------- | ------------------------------------------------------- |
| `index`  | ✅       | 表示要删除的别名的编号                                  |
| `alias`  | ✅       | 表示别名名词。不能为纯数字，最长 15 字符                |
| `player` | ✅       | 可输入游戏 UID，或使用格式 `服务器 IGN`（空格分隔）绑定 |

### 📌 使用示例

```bash
# 查看当前别名列表
/alias list

# 删除第 2 个别名
/alias del 2

# 添加别名，绑定 UID
/alias add myfriend 2023619512

# 添加别名，绑定服务器+IGN
/alias add captain asia SangonomiyaKokomi_
```

### 💡 注意事项

- 别名长度不能超过 15 个字符
- 别名 **不能为纯数字**，以避免和 UID 混淆。

## `/algo` 指令说明

**用于切换用户评分算法。**

目前支持两种算法：

- `default`：默认评分算法。
- `none`：关闭评分功能，不再显示评分。

### 🧩 指令格式

```bash
/algo mode:<评分算法>
```

### 🔧 参数说明

| 参数名 | 是否必填 | 说明                         |
| ------ | -------- | ---------------------------- |
| `mode` | ✅       | 可选值为 `default` 或 `none` |

### 📌 使用示例

```bash
/algo default
/algo none
```

## `/mode` 指令说明

**用于切换 Bot 返回图片的显示模式（日间 / 夜间）。**

目前支持两种模式：

- `light`：日间模式，亮色背景。
- `dark`：夜间模式，暗色背景。

### 🧩 指令格式

```bash
/mode mode:<显示模式>
```

### 🔧 参数说明

| 参数名 | 是否必填 | 说明                       |
| ------ | -------- | -------------------------- |
| `mode` | ✅       | 可选值为 `light` 或 `dark` |

### 📌 使用示例

```bash
/mode dark
/mode light
```

---

## `/theme` 指令说明

**用于切换 Bot 返回图片的视觉主题风格，或查看可用主题列表。**

当前支持两种用法：

1. 使用 `list` 参数查看所有可选主题及其代码。
2. 指定主题代码以切换当前主题。

### 🧩 指令格式

```bash
/theme list
# 或
/theme theme:<主题代码>
```

### 🔧 参数说明

| 参数名  | 是否必填 | 说明                                   |
| ------- | -------- | -------------------------------------- |
| `theme` | ✅       | 输入主题代码, 使用 `list` 查看可选主题 |

### 📌 使用示例

```bash
# 查看当前可选主题列表
/theme list

# 切换为主题 mygo
/theme mygo
```

## `/stat` 指令说明

**用于查看指定用户的战绩统计信息。支持简略展示，并可通过参数查询特定模式的战绩。**

### 🧩 指令格式

```bash
/stat player:[用户参数] mode:[模式参数]
```

### 🔧 参数说明

| 参数名   | 是否必填 | 说明                                                                                                                                                  |
| -------- | -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| `player` | 可选     | 用于指定要查询的游戏账号，支持以下四种方式：<br>1. `服务器 IGN`（空格分隔）<br>2. 游戏 UID<br>3. @某个已绑定账号的用户<br>4. 使用 `/alias` 设置的别名 |
| `mode`   | 可选     | 查询指定模式的战绩数据。若省略则默认返回总体战绩数据。详见下方支持的模式列表。                                                                        |

#### 支持的模式参数

| 模式参数名 | 说明                                 |
| ---------- | ------------------------------------ |
| `pvp`      | 随机战数据                           |
| `rank`     | 排位战数据                           |
| `solo`     | 单野（随机）数据                     |
| `div2`     | 双排（随机）数据                     |
| `div3`     | 三排（随机）数据                     |
| `cv`       | 航母（随机）数据                     |
| `bb`       | 战列舰（随机）数据                   |
| `ca`       | 巡洋舰（随机）数据                   |
| `dd`       | 驱逐舰（随机）数据                   |
| `ss`       | 潜艇（随机）数据                     |
| `surface`  | 水面舰艇（随机）（不包含航母和潜艇） |

### 📌 使用示例

```bash
# 示例1：查询自己绑定账号的总体战绩
/stat

# 示例2：查询自己绑定账号的 pvp 模式战绩
/stat pvp

# 示例3：通过 IGN 查询他人的总体战绩
/stat asia SangonomiyaKokomi_

# 示例4：通过 UID 查询
/stat 2023123456

# 示例5：通过 @某用户 查询其绑定账号的总体战绩
/stat @Someone

# 示例6：通过别名查询（假设设置了别名 myfriend）
/stat myfriend

# 示例7：查询指定用户的 rank 模式战绩
/stat asia SangonomiyaKokomi_ rank
```

### 💡 注意事项

- 参数顺序固定，**用户参数必须在前，模式参数在后**
- 最多只能传入一个 `用户参数` 和 `模式参数`
