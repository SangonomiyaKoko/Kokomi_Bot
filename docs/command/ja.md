```bash
/cls
```

### 🔧 パラメータ説明

なし

### 📌 使用例

```bash
/cls
```

- この操作は元に戻せません。慎重に使用してください。

## `/link` コマンド説明

**ゲームアカウントを連携するためのコマンドです。UID または「サーバー名 + IGN」で登録できます。**

### 🧩 コマンド形式

```bash
/link user:<ユーザー識別子>
```

### 🔧 パラメータ説明

| パラメータ | 必須 | 説明                                                                 |
| ---------- | ---- | -------------------------------------------------------------------- |
| `user`     | ✅   | UID または `サーバー IGN` 形式（スペースで区切る）で入力してください |

### 📌 使用例

```bash
# 例1：UIDで連携
/link 2023619512

# 例2：サーバー名とIGNで連携
/link asia SangonomiyaKokomi_
```

## `/lang` コマンド説明

**Bot の表示言語を切り替えるためのコマンドです。現在は中国語（cn）、英語（en）、日本語（ja）をサポートしています。**

### 🧩 コマンド形式

```bash
/lang lang:<言語コード>
```

### 🔧 パラメータ説明

| パラメータ | 必須 | 説明                                                                 |
| ---------- | ---- | -------------------------------------------------------------------- |
| `lang`     | ✅   | 使用可能：`cn`、`en`、`ja` |

### 📌 使用例

```bash
/lang en
```

## `/algo` コマンド説明

**ユーザーの評価アルゴリズムを切り替えるためのコマンドです。**

現在サポートされているモードは次の通りです：

- `default`：デフォルトの評価アルゴリズム。
- `none`：評価を無効にする（表示しない）。

### 🧩 コマンド形式

```bash
/algo mode:<評価モード>
```

### 🔧 パラメータ説明

| パラメータ | 必須 | 説明                                                                 |
| ---------- | ---- | -------------------------------------------------------------------- |
| `mode`     | ✅   | 指定可能：`default`、`none` |

### 📌 使用例

```bash
/algo default
/algo none
```

## `/mode` コマンド説明

**Botが返す画像の表示モード（昼 / 夜）を切り替えるためのコマンドです。**

現在サポートされているモード：

- `light`：昼モード（明るい背景）
- `dark`：夜モード（暗い背景）

### 🧩 コマンド形式

```bash
/mode mode:<表示モード>
```

### 🔧 パラメータ説明

| パラメータ | 必須 | 説明                                                                 |
| ---------- | ---- | -------------------------------------------------------------------- |
| `mode`     | ✅   | 指定可能：`light` または `dark` |

### 📌 使用例

```bash
/mode dark
/mode light
```

## `/theme` コマンド説明

**Botの画像スタイルテーマを切り替えたり、利用可能なテーマ一覧を確認したりできます。**

以下の2つの使い方があります：

1. `list` を指定して利用可能なテーマ一覧を表示する
2. テーマコードを指定してテーマを切り替える

### 🧩 コマンド形式

```bash
/theme list
# または
/theme theme:<テーマコード>
```

### 🔧 パラメータ説明

| パラメータ | 必須 | 説明                                                                 |
| ---------- | ---- | -------------------------------------------------------------------- |
| `theme`     | ✅   | テーマコードを入力、または `list` で一覧を表示 |

### 📌 使用例

```bash
/theme list
/theme mygo
```
