## `/cls` コマンド説明

```bash
/cls
```

### 🔧 パラメータ説明

なし

```bash
/cls
```

- この操作は元に戻せません。慎重に使用してください。

## `/link` コマンド説明

**ゲームアカウントを連携するためのコマンドです。UID または「サーバー名 + IGN」で登録できます。**

### 🧩 コマンド形式

```bash
/link user:<ユーザー識別子>
```

### 🔧 パラメータ説明

| パラメータ | 必須 | 説明                                                                 |
| ---------- | ---- | -------------------------------------------------------------------- |
| `user`     | ✅   | UID または `サーバー IGN` 形式（スペースで区切る）で入力してください |

### 📌 使用例

```bash
# 例1：UIDで連携
/link 2023619512

# 例2：サーバー名とIGNで連携
/link asia SangonomiyaKokomi_
```

## `/lang` コマンド説明

**Bot の表示言語を切り替えるためのコマンドです。現在は中国語（cn）、英語（en）、日本語（ja）をサポートしています。**

### 🧩 コマンド形式

```bash
/lang lang:<言語コード>
```

### 🔧 パラメータ説明

| パラメータ | 必須 | 説明                       |
| ---------- | ---- | -------------------------- |
| `lang`     | ✅   | 使用可能：`cn`、`en`、`ja` |

### 📌 使用例

```bash
/lang en
```

## `/algo` コマンド説明

**ユーザーの評価アルゴリズムを切り替えるためのコマンドです。**

現在サポートされているモードは次の通りです：

- `default`：デフォルトの評価アルゴリズム。
- `none`：評価を無効にする（表示しない）。

### 🧩 コマンド形式

```bash
/algo mode:<評価モード>
```

### 🔧 パラメータ説明

| パラメータ | 必須 | 説明                        |
| ---------- | ---- | --------------------------- |
| `mode`     | ✅   | 指定可能：`default`、`none` |

### 📌 使用例

```bash
/algo default
/algo none
```

## `/mode` コマンド説明

**Bot が返す画像の表示モード（昼 / 夜）を切り替えるためのコマンドです。**

現在サポートされているモード：

- `light`：昼モード（明るい背景）
- `dark`：夜モード（暗い背景）

### 🧩 コマンド形式

```bash
/mode mode:<表示モード>
```

### 🔧 パラメータ説明

| パラメータ | 必須 | 説明                            |
| ---------- | ---- | ------------------------------- |
| `mode`     | ✅   | 指定可能：`light` または `dark` |

### 📌 使用例

```bash
/mode dark
/mode light
```

## `/theme` コマンド説明

**Bot の画像スタイルテーマを切り替えたり、利用可能なテーマ一覧を確認したりできます。**

以下の 2 つの使い方があります：

1. `list` を指定して利用可能なテーマ一覧を表示する
2. テーマコードを指定してテーマを切り替える

### 🧩 コマンド形式

```bash
/theme list
# または
/theme theme:<テーマコード>
```

### 🔧 パラメータ説明

| パラメータ | 必須 | 説明                                           |
| ---------- | ---- | ---------------------------------------------- |
| `theme`    | ✅   | テーマコードを入力、または `list` で一覧を表示 |

### 📌 使用例

```bash
/theme list
/theme mygo
```

## `/stat` コマンド説明

**指定したユーザーの戦績を確認します。簡易表示に対応しており、モード指定も可能です。**

### 🧩 コマンド形式

```bash
/stat player:[ユーザー情報] mode:[モード情報]
```

### 🔧 パラメータ説明

| パラメータ | 必須 | 説明                                                                                                                                                           |
| ---------- | ---- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `player`   | 任意 | 対象アカウントを指定します。以下の 4 つの方法に対応：<br>1. `サーバー IGN`（スペース区切り）<br>2. UID<br>3. @で他ユーザーを指定<br>4. `/alias` で設定した別名 |
| `mode`     | 任意 | 指定モードの戦績を取得します。省略すると全体戦績が表示されます。対応モードは下記参照。                                                                         |

#### 対応モード一覧

| モード名       | 説明                                   |
| -------------- | -------------------------------------- |
| `random`       | ランダム戦                             |
| `ranked`       | ランク戦                               |
| `solo`         | ソロ出撃（ランダム）                   |
| `div2`         | 2 人分艦隊（ランダム）                 |
| `div3`         | 3 人分艦隊（ランダム）                 |
| `AirCarrier`   | 空母（ランダム）                       |
| `Battleship`   | 戦艦（ランダム）                       |
| `Cruiser`      | 巡洋艦（ランダム）                     |
| `Destroyer`    | 駆逐艦（ランダム）                     |
| `Submarine`    | 潜水艦（ランダム）                     |
| `SurfaceShips` | 水上艦（空母と潜水艦を除くランダム戦） |

### 📌 使用例

```bash
# 例1：自分の全体戦績を確認
/stat

# 例2：自分の PVP 戦績を確認
/stat pvp

# 例3：IGNで他人の戦績を確認
/stat asia SangonomiyaKokomi_

# 例4：UIDで検索
/stat 2023123456

# 例5：@ユーザーでバインド済みアカウントの戦績を確認
/stat @Someone

# 例6：別名で検索（例："myfriend"）
/stat myfriend

# 例7：指定ユーザーのランク戦戦績を確認
/stat asia SangonomiyaKokomi_ rank
```

### 💡 注意事項

- パラメータの順序は固定です：**ユーザー → モードの順**
- 指定できるのは `ユーザー情報` と `モード情報` の各 1 つまで
